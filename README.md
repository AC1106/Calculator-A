##簡易計算機
這是一個基於命令列的簡易計算機程式，使用C#語言編寫。
它支援基本的算術運算，包括加法、減法、乘法、除法、平方根、指數運算，以及複數加法和減法。


##功能
加法：將兩個數字相加。
減法：從第一個數字中減去第二個數字。
乘法：將兩個數字相乘。
除法：將第一個數字除以第二個數字。
平方根：計算給定數字的平方根。
指數運算：計算給定基數和指數的冪。
複數加法：將兩個複數相加。
複數減法：從第一個複數中減去第二個複數。
儲存到記憶：將數字儲存到記憶中。
從記憶讀取：從記憶中讀取儲存的數字。
清除記憶：清除記憶中的數字。
查看歷史記錄：顯示過去的計算歷史記錄。


##使用方法
在命令列中選擇所需的操作。
根據提示輸入相關數字。
程式將根據您的輸入執行選定的操作，並在螢幕上顯示結果。
